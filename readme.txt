Easy to install.2. Setting up Auto Mailchimp Popup is very easy.3. You can customize CSS easily.4. Can display popup in Homepage only. Two Options: Homepage or All Pages.5. Well Documented.== Installation ==This section describes how to install the plugin and get it working.1. Install the wordpress plugin firstYou can install it via Wordpress Installer or can download the file first then Upload it on 'wp-content' folder2. Activate the plugin3. You will see "Mailchimp" settings from right side of your wordpress menu.4. You will need to create a Mailchimp API and input your list id on settings area.Read more about creating API and Search List ID here: http://atomixstudios.com/mailchimpdemo/documentation/== Screenshots ==1.
